# Recipe Book

A responsive recipe book website with categories, featured recipes, step-by-step cooking guides, chef ratings, and a form to add your own recipes.

## Features
- Category navigation and quick filters
- Search and sort recipes
- Recipe cards with images and details
- Modal with ingredients and step-by-step guide
- Add-your-own recipe form (instant preview)
- Rate recipes and view chef ratings
- Mobile-friendly navigation

## Files
- [index.html](index.html) — Page structure and content
- [styles.css](styles.css) — Styling and layout
- [app.js](app.js) — Data, rendering, and interactions

## How to Use
1. Open [index.html](index.html) in a browser.
2. Browse categories and featured recipes.
3. Click “View Recipe” to see steps and rate.
4. Use the form to add a new recipe.

## Customization
- Add more categories and recipes in [app.js](app.js).
- Update colors and typography in [styles.css](styles.css).
